📂 Pretty-ls

A human-friendly and emoji-enhanced ls replacement for the terminal

📱 Cross-Platform 🐚 Multi-Shell ⚡ Fast & Lightweight 🎨 Beautiful Output
Get Started View on GitHub

✨ Features

📁

Emoji Icons

Folders and files are distinguished with clear emoji icons for instant visual recognition.

🎨

Color Coding

Different file types are color-coded for easy identification - shell scripts, images, documents, and more.

📏

Smart Truncation

Long filenames are intelligently truncated with ellipsis to maintain clean, readable output.

📊

Human-Readable Sizes

File sizes displayed in B, KB, MB, GB as appropriate for easy understanding at a glance.

🕒

Modification Dates

Shows when files were last modified in a clear, consistent format across all platforms.

🖥️

Cross-Platform

Works seamlessly on macOS, Linux, and Windows with native shell integration.

🐚

Multi-Shell Support

Supports Bash, Zsh, Fish, and PowerShell with automatic detection and configuration.

⚡

Fast & Lightweight

Minimal dependencies and optimized performance for instant directory listings.

🔧

Easy Installation

One-command installation for each platform with automatic PATH configuration.

$ pls

🚀 Installation

curl -sSL https://raw.githubusercontent.com/KoDesigns/pretty-ls/main/install/install-macos.sh | bash

# Supports macOS 10.12+ with Bash 4.0+ or Zsh 5.0+
# Automatically configures your shell and PATH
curl -sSL https://raw.githubusercontent.com/KoDesigns/pretty-ls/main/install/install-linux.sh | bash

# Supports any modern Linux distribution
# Works with Bash, Zsh, and Fish shells
# Automatically installs dependencies if needed
iwr -useb https://raw.githubusercontent.com/KoDesigns/pretty-ls/main/install/install-windows.ps1 | iex

# Requires Windows 10+ with PowerShell 5.1+
# Creates a PowerShell module and updates your profile
# Run in PowerShell as Administrator if needed

Need help? Check out our detailed installation guide

After installation, restart your terminal or run: source ~/.zshrc
